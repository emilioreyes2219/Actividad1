# Actividad 1 - Ejercicios Web

Este repositorio contiene 10 ejercicios de HTML, CSS y JavaScript.
## Ejercicio 1: Hola Mundo
Se creó una página básica con el mensaje "Hola Mundo" para comprobar la conectividad con XAMPP y el funcionamiento de HTML.

## Ejercicio 2: Párrafos y Formato
Se utilizaron párrafos y etiquetas de formato para dar estructura y resaltar el contenido del texto.

## Ejercicio 3: Enlaces
Se crearon enlaces para navegar a otras páginas web y practicar el uso de hipervínculos.

## Ejercicio 4: Imágenes
Se agregaron imágenes a la página utilizando rutas y atributos para mostrarlas correctamente.

## Ejercicio 5: Texto y Enlaces
Se combinaron textos, imágenes y enlaces en una misma página para integrar los elementos aprendidos.

## Ejercicio 6: Secciones Semánticas
Se organizó el contenido usando etiquetas semánticas como `header`, `section` y `footer`.

## Ejercicio 7: Listas Anidadas
Se creó una receta utilizando listas ordenadas y desordenadas para organizar la información.

## Ejercicio 8: Horario de Clases
Se elaboró un horario semanal utilizando tablas para organizar materias y horarios.

## Ejercicio 9: Formulario de Contacto
Se diseñó un formulario de contacto con diferentes tipos de campos y validaciones básicas.

## Ejercicio 10: Conversor de Temperatura
Se desarrolló un conversor que transforma grados Celsius a grados Fahrenheit mediante JavaScript.
